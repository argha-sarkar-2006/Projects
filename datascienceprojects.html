Sentiment Analysis

Analyze emotions in text data using NLP and machine learning.

Python NLTK Scikit-learn
sentiment_analysis.py 
import pandas as pd
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.svm import SVC

# Load dataset
data = pd.read_csv('reviews.csv')

# Vectorize text
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(data['text'])

# Train model
model = SVC(kernel='linear')
model.fit(X, data['sentiment'])

# Predict sentiment
def predict_sentiment(text):
    return model.predict(vectorizer.transform([text]))

1. Install required libraries: `pip install pandas scikit-learn nltk`
2. Prepare your dataset in CSV format with 'text' and 'sentiment' columns.
3. Run the script to train the model.
4. Use `predict_sentiment()` to analyze new text.

House Price Prediction

Predict housing prices using regression models.

Python Pandas XGBoost
house_price.py 
import xgboost as xgb
from sklearn.model_selection import train_test_split

# Load data
data = pd.read_csv('housing.csv')

# Feature engineering
X = data.drop('price', axis=1)
y = data['price']

# Split data
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Train model
model = xgb.XGBRegressor()
model.fit(X_train, y_train)

# Evaluate
score = model.score(X_test, y_test)

1. Install XGBoost: `pip install xgboost`
2. Prepare your dataset with features like square footage, bedrooms, etc.
3. Run the script to train the model.
4. Use `model.predict()` for new predictions.

Customer Segmentation

Cluster customers using K-Means for targeted marketing.

Python Scikit-learn Matplotlib
segmentation.py 
from sklearn.cluster import KMeans
import matplotlib.pyplot as plt

# Load customer data
data = pd.read_csv('customers.csv')

# Select features
X = data[['age', 'income', 'spending_score']]

# Apply K-Means
kmeans = KMeans(n_clusters=5)
clusters = kmeans.fit_predict(X)

# Visualize
plt.scatter(X['income'], X['spending_score'], c=clusters)
plt.show()

1. Install required libraries: `pip install scikit-learn matplotlib`
2. Prepare customer data with relevant features.
3. Run the script to segment customers.
4. Adjust `n_clusters` for different groupings.

Recommendation System

Build a movie recommendation engine using collaborative filtering.

Python Surprise Pandas
recommendation.py 
from surprise import Dataset, KNNBasic
from surprise.model_selection import train_test_split

# Load data
data = Dataset.load_builtin('ml-100k')

# Train-test split
trainset, testset = train_test_split(data, test_size=0.2)

# Train model
model = KNNBasic()
model.fit(trainset)

# Predict
predictions = model.test(testset)

1. Install Surprise: `pip install scikit-surprise`
2. Use MovieLens dataset or your own ratings data.
3. Train the model to find similar users/items.
4. Generate recommendations with `model.predict()`.
